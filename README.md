# Online Toy Store

One unified toy store application with a single user flow.

## Customer flow

1. **Shop** — open http://localhost:8080/ (browse toys)
2. **Buy Now** — pick a toy → login if needed → checkout
3. **Place Order** — confirm address and payment method
4. **Pay** — complete card details after order
5. **Review** — leave a review from the toy card or Reviews page

All pages share the same top navigation (Shop, Orders, Payments, Reviews, Profile).

## Admin flow

- Admin login: `/admin/login.jsp`
- Dashboard: manage admins, manage toys (inventory)

## Run

```powershell
cd "c:\Users\USER\Downloads\Toy store"
.\run.ps1
```

Open **http://localhost:8080/**

## Build

```powershell
.\mvnw.cmd clean package
```
